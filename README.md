A quick converter from hex to RGB

## Getting Started

Install it via npm:

```shell
npm install @clef/hex-to-rgb
```

And include in your project:

```javascript
import hexToRGB from '@clef/hex-to-rgb';
```

## Usage

### hexToRGB(hexString)

Converts a hex color string to a RGB object
